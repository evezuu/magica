# Magica Maxxing

Hello! This was meant to be an interactive game project made at Outpost. Unfortunately, we weren't able to get the mechanics of it working in time. 

The main premise of the game was that it would be a sort of wizard duel. There would be two wands, and they would track motion using BMI160 IMUs. People would be able to cash spells with this wand and defeat their opponents.

We also had a lot of other game mode ideas we could've done for this, such as a "rhythm game" mode where people had to do stuff based of constant instructions on screen, and a single-player mode inspired by the google Halloween Cat game. 

Unfortunately, even after working for several hours straight, we were unable to get the BMIs working. We did ask a lot of people for help with it, but they were unable to make them work too. Because of severe time constraints, we had to rush the wiring for the other components of our proejcts (we wanted to add LEDs and a button). However, we were pretty inexperienced and we accidentally wired out items wrong, causing the project to fail.

A staff member did help us make the project light up again, so it can do that still.

For outpost, we're thinking of using the 3D-printed wands we have, and we're gonna make some software that allows people to use the wands for a photobooth of sorts.

In the future, I will take care to acknowledge the time-constraints we have. This event has taught me so much anout hardware, and I am glad to have experienced it.
